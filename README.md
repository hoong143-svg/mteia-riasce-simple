# MTEIA RIASEC Simple

简體中文 + English 版本

## 主要区别

- 语言：简體中文 (zh-CN) + English
- 默认语言：zh-CN

## 与 mteia-riasce 的区别

| 功能 | mteia-riasce | mteia-riasce-simple |
|------|--------------|---------------------|
| 默认语言 | 繁体中文 (zh-TW) | 简体中文 (zh-CN) |
| 语言选项 | 繁体/English | 简体/English |

## 本地开发

```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend ( separate terminal)
cd backend
npm install
node server.js
```

## 部署

使用 `render.yaml` 在 Render 上部署
